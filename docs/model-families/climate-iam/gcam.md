> A market-driven, technology-rich partial-equilibrium IAM coupling economy, energy, land, water and climate across 32 regions; solves for prices that clear each market each period.

## Positioning card

| Axis (see [Taxonomy](../../foundations/taxonomy.md)) | GCAM |

| Optimization vs Simulation | Simulation via market clearing |
| Top-down vs Bottom-up | Hybrid (bottom-up tech + top-down) |
| Equilibrium | Partial equilibrium |
| Foresight | Recursive-dynamic (myopic) |
| Deterministic vs Stochastic | Deterministic (+ scenario/UQ) |
| Time / Space | 5-yr / 32 regions + land grid |
| Solution method | Iterative price search (partial equilibrium) |
